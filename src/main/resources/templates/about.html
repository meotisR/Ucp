О проекте

Это веб-сайт, который облегчает процесс заказа перевозки грузов для пользователей и предоставляет возможность создания профилей для логистических компаний. Это позволяет клиентам быстро находить подходящих перевозчиков, а компании - эффективно управлять своей деятельностью.

Use cases:

  • Для обычных пользователей:
    • Авторизация/регистрация
    • Редактрование профиля
    • Создание, редактирование компании
    • Создание создание заказов
    • Выполнение заказов
    • Пополнение профиля
